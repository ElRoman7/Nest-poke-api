## Project setup

```bash
$ yarn install
```

## Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar

```
$ yarn install
```

3. Tener el cli instalado

```
$ npm i -g @nest/cli
```

4. Levantar la Base de Datos (Usando Docker)

```
$ docker-composer up -d
```

## Stack Usado

- MongoDB
- Nest
- Angular
